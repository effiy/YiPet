# LoadingSpinner 组件

> 温柔陪伴助手 - 加载动画组件

---

## 概述

LoadingSpinner 是温柔陪伴助手的加载动画组件，用于显示加载状态。

---

## 文件位置

```
modules/pet/components/通用组件/LoadingSpinner/
├── index.html         # 组件模板
└── index.js           # 组件逻辑
```

---

## 功能特性

- **加载动画**：显示加载动画
- **尺寸控制**：支持不同尺寸的加载动画
- **颜色控制**：支持不同颜色的加载动画
