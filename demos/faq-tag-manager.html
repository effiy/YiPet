FaqTagManager 使用事例

概述

FaqTagManager 是 FAQ 标签编辑子弹窗，由 FaqManager.editTags() 触发。支持手动输入添加标签、快速标签一键添加、智能生成标签、保存更新。

状态管理

const store = reactive({
  visible: false,
  inputValue: '',
  currentTags: [],      // 当前 FAQ 项的标签数组
  faqIndex: -1,          // 正在编辑的 FAQ 索引
})

// 快速标签 — 预定义的常用标签
const QUICK_TAGS = ['工具', '开源项目', '家庭', '工作', '娱乐', '文档', '日记']

// 方法
close()           // 关闭弹窗
addFromInput()    // 从输入框添加标签（自动 trim + 去重）
addQuickTag(tag)   // 点击快速标签添加
removeTag(tag)    // 移除指定标签
save()            // 保存标签更新到 FAQ 数据源
smartGenerate()   // AI 智能生成标签

方法详解

addFromInput — IME 安全 + 去重

addFromInput() {
  const value = store.inputValue.trim()
  if (!value) return
  if (store.currentTags.some(t => t.toLowerCase() === value.toLowerCase())) return
  store.currentTags.push(value)
  store.inputValue = ''
}

// smartGenerate: 调用 AI 根据 FAQ 内容分析并返回建议标签
async smartGenerate() {
  const faq = store.allFaqs?.[store.faqIndex]
  const tags = await manager.generateTagsForFaq(faq)  // AI 调用
  store.currentTags = [...new Set([...store.currentTags, ...tags])]
}
