SessionTagManager 使用事例

概述

SessionTagManager 是会话标签管理弹窗，支持拖拽排序、快速标签、手动输入添加、IME 输入安全、保存更新会话标签。

状态管理

Store + 拖拽状态

const store = reactive({
  sessionId: null,
  currentTags: [],         // 当前会话的标签数组
  quickTags: [],           // 从 manager.getAllTags() 获取的全局标签
  draggingIndex: -1,       // 正在拖拽的标签索引
  dragOverIndex: -1,      // 拖拽悬停的标签索引
  dragOverPosition: 'before' // 悬停位置: 'before' | 'after'
})

// 方法
close() · addFromInput() · addQuickTag(tag) · removeTag(tag) · save()
reorderTag(fromIndex, toIndex)  // 拖拽后重新排序

// 拖拽事件 (HTML5 Drag API)
onDragStart(index, e) · onDragEnd(e)
onDragOver(index, e) · onDragLeave(e) · onDrop(index, e)

拖拽排序算法

onDrop(targetIndex, e) {
  const from = store.draggingIndex
  const rect = e.target.getBoundingClientRect()
  const midY = rect.top + rect.height / 2

  // 判断插入位置: 上半部 → before, 下半部 → after
  const insertPos = e.clientY < midY ? 'before' : 'after'
  const to = insertPos === 'before' ? targetIndex : targetIndex + 1

  // 实际插入位置需要调整 (移除原位置后的偏移)
  const actualTo = from < to ? to - 1 : to
  reorderTag(from, actualTo)
}
