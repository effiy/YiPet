# 虚拟宠物展示

> 在网页上显示可爱的虚拟宠物

---

## 功能概述

虚拟宠物展示功能在网页右下角（或用户自定义位置）显示可爱的虚拟宠物角色，为用户提供陪伴感。

---

## 主要特性

### 1. 宠物显示
- 在网页上显示可爱的虚拟宠物图标
- 支持多种宠物角色（教师、医生、甜品师、警察）
- 自适应不同网页环境

### 2. 拖拽移动
- 支持点击并拖拽宠物到页面任意位置
- 自动记忆宠物位置，下次打开页面时保持
- 流畅的拖拽动画效果

### 3. 动画效果
- 宠物浮动动画
- 眨眼效果
- 呼吸动画
- 根据状态切换不同动画

---

## 使用说明

### 显示/隐藏宠物
- **方式一**：点击浏览器工具栏扩展图标，在弹出面板中切换显示状态
- **方式二**：使用快捷键 `Ctrl+Shift+P` (Mac: `Cmd+Shift+P`)

### 移动宠物
1. 将鼠标悬停在宠物图标上
2. 按住鼠标左键
3. 拖动到目标位置
4. 松开鼠标完成移动

---

## 技术实现

### 相关文件
- `modules/pet/content/petManager.pet.js` - 宠物显示和动画管理
- `modules/pet/content/petManager.drag.js` - 拖拽功能实现
- `modules/pet/content/petManager.state.js` - 状态管理
- `assets/styles/content.css` - 宠物样式

### 存储数据
宠物位置和显示状态存储在 Chrome 存储中：
```javascript
petGlobalState: {
  visible: true,              // 宠物是否可见
  position: { x: 100, y: 100 }, // 宠物位置坐标
  size: { width: 80, height: 80 } // 宠物大小
}
```

---

## 自定义配置

### 调整宠物大小
通过修改 `petManager.pet.js` 中的配置项：
```javascript
const PET_CONFIG = {
  width: 80,  // 宠物宽度
  height: 80  // 宠物高度
};
```

### 添加新动画
在 `assets/styles/base/animations.css` 中添加新的 CSS 动画。
