# 构建与部署

> 本文档描述项目的构建、部署和运维流程。

## 构建流程

### 环境要求

- Chrome 浏览器 >= 88（支持 Manifest V3）

### 开发构建

本项目采用零构建架构，无需构建工具，直接加载源代码到 Chrome 即可运行。

开发步骤：

1. 克隆仓库到本地
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"按钮
5. 选择仓库目录
6. 打开任意网页测试扩展功能

## 运维监控

### 扩展状态检查

在 chrome://extensions/ 页面可以查看：
- 查看扩展是否启用状态
- 查看错误日志
- 重新加载扩展
- 检查权限审查扩展

### 日志查看

- Content Script 日志：在网页中打开 DevTools Console 查看
- Background Script 日志：在 chrome://extensions/ 中点击"Service Worker"查看
- Popup 日志：右键点击扩展图标 -> "检查弹出内容"
