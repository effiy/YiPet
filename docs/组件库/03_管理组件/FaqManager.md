# FaqManager 组件

> 温柔陪伴助手 - FAQ 管理器组件

---

## 概述

FaqManager 是温柔陪伴助手的 FAQ 管理器组件，用于管理常见问题和答案。

---

## 文件位置

```
modules/pet/components/manager/FaqManager/
├── index.html         # 组件模板
└── index.js           # 组件逻辑
```

---

## 功能特性

- **FAQ 列表**：显示 FAQ 列表
- **添加 FAQ**：支持添加新的 FAQ
- **编辑 FAQ**：支持编辑 FAQ
- **删除 FAQ**：支持删除 FAQ
