🐾 Chrome扩展安装指南

🎉 恭喜！ 你已经成功下载了可拖拽小宠物Chrome扩展。按照以下步骤完成安装：

1准备图标文件

首先需要生成扩展所需的图标文件：

  1. 打开 generate-icons.html 文件
  2. 在浏览器中运行该文件
  3. 点击下载按钮获取各个尺寸的PNG图标
  4. 将下载的图标文件放入 icons/ 目录

2打开Chrome扩展管理页面

在Chrome浏览器中打开扩展管理页面：

chrome://extensions/

或者通过菜单：菜单 → 更多工具 → 扩展程序

3启用开发者模式

在扩展管理页面的右上角，找到"开发者模式"开关并启用它：

⚠️ 注意： 开发者模式允许安装未打包的扩展，请确保只安装可信的扩展。

4加载扩展

点击"加载已解压的扩展程序"按钮：

  1. 选择包含扩展文件的 chrome-extension 文件夹
  2. 点击"选择文件夹"
  3. 扩展将出现在扩展列表中

5完成安装

确保扩展已启用（开关为蓝色），然后：

  1. 刷新任意网页
  2. 可爱的小宠物将出现在页面上
  3. 点击扩展图标可以控制宠物
🎊 安装完成！ 现在你可以在任何网页上享受与可爱宠物的互动了！

🔧 故障排除：
  • 如果宠物不显示，请刷新网页
  • 如果无法拖拽，请检查是否有其他脚本冲突
  • 如果设置不保存，请检查Chrome存储权限
  • 如果图标不显示，请确保icons目录中有PNG文件
