# ChatInput 组件

> 温柔陪伴助手 - 聊天输入组件

---

## 概述

ChatInput 是温柔陪伴助手的聊天输入组件，提供文本输入功能。

---

## 文件位置

```
modules/pet/components/chat/ChatInput/
├── index.html         # 组件模板
└── index.js           # 组件逻辑
```

---

## 功能特性

- **文本输入**：提供文本输入框
- **发送按钮**：支持点击发送消息
- **自动高度**：输入框高度自动调整
