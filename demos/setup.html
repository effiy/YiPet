setup.mjs 使用事例

概述

setup.mjs 是 vitest 全局初始化文件（setupFiles），在所有测试前执行。提供 chrome API mock、IIFE 模块加载器、状态重置、生命周期钩子。

setup.mjs 结构

// tests/setup.mjs — vitest setupFiles
import { beforeEach, afterEach, vi } from 'vitest';

// 1. chrome.storage.local Map mock
const storage = new Map();
globalThis.chrome = {
  storage: {
    local: {
      get(keys) { ... },
      set(items) { ... },
      remove(keys) { ... },
      clear() { storage.clear(); },
    }
  },
  runtime: { getURL(p) { return 'chrome-extension://id/' + p; } }
};

// 2. fetch mock
globalThis.fetch = vi.fn();

// 3. loadModule — IIFE 模块加载器
globalThis.loadModule = async (path) => { ... };

// 4. 生命周期 — 状态重置
beforeEach(() => {
  storage.clear();
  globalThis.fetch.mockReset();
});

afterEach(() => {
  vi.restoreAllMocks();
});

chrome API Mock

Map 实现 chrome.storage.local 完整接口。

chrome.storage.local 模拟

// get — 支持单键、数组键、null(取全部)
async function get(keys) {
  const result = {};
  if (keys === null) { storage.forEach((v,k) => result[k] = v); }
  else if (Array.isArray(keys)) { keys.forEach(k => { if (storage.has(k)) result[k] = storage.get(k); }); }
  else if (storage.has(keys)) { result[keys] = storage.get(keys); }
  return result;
}
