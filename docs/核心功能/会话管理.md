# 会话管理

> 保存和管理多个对话会话

---

## 功能概述

会话管理功能允许用户创建、保存、切换和删除多个对话会话，帮助用户整理不同主题的聊天记录。

---

## 主要特性

### 1. 会话创建
- 创建新的对话会话
- 自定义会话名称和描述
- 支持添加会话标签

### 2. 会话切换
- 快速切换不同会话
- 会话列表管理
- 当前会话状态显示

### 3. 会话编辑
- 修改会话信息
- 编辑会话标签
- 更新会话内容

### 4. 标签管理
- 创建和管理会话标签
- 给会话添加多个标签
- 按标签筛选会话

---

## 使用说明

### 创建新会话
1. 打开聊天窗口
2. 点击会话列表按钮
3. 选择"新建会话"
4. 输入会话名称和描述
5. 点击确认创建

### 切换会话
1. 打开会话列表
2. 点击要切换的会话
3. 自动加载该会话的聊天记录

### 编辑会话
1. 在会话列表中选择会话
2. 点击编辑按钮
3. 修改会话信息
4. 保存更改

## 技术实现

### 相关文件
- `modules/pet/content/petManager.session.js` - 会话管理模块
- `modules/session/` - 会话管理功能模块
- `modules/pet/components/editor/SessionInfoEditor/` - 会话信息编辑器
- `core/api/services/SessionService.js` - 会话 API 服务

### 数据结构
```javascript
{
  id: 'session-id',
  name: '会话名称',
  description: '会话描述',
  tags: ['标签1', '标签2'],
  messages: [...],
  createdAt: Date.now(),
  updatedAt: Date.now(),
  isFavorite: false
}
```

### 存储位置
- 本地存储：Chrome `storage.local` 中的 `petSessions` 键
- 云端同步：通过 API 服务同步（如果配置）
