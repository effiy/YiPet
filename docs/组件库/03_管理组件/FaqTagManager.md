# FaqTagManager 组件

> 温柔陪伴助手 - FAQ 标签管理器组件

---

## 概述

FaqTagManager 是温柔陪伴助手的 FAQ 标签管理器组件，用于管理 FAQ 的标签。

---

## 文件位置

```
modules/pet/components/manager/FaqTagManager/
├── index.html         # 组件模板
└── index.js           # 组件逻辑
```

---

## 功能特性

- **标签列表**：显示标签列表
- **添加标签**：支持添加新标签
- **删除标签**：支持删除标签
- **标签分配**：支持为 FAQ 分配标签
