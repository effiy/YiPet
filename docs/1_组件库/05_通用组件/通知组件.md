# Notification 组件

> 温柔陪伴助手 - 通知组件

---

## 概述

Notification 是温柔陪伴助手的通知组件，用于显示通知消息。

---

## 文件位置

```
modules/pet/components/通用组件/Notification/
├── index.html         # 组件模板
└── index.js           # 组件逻辑
```

---

## 功能特性

- **通知显示**：显示通知消息
- **类型支持**：支持不同类型的通知（成功、错误、警告）
- **自动关闭**：支持通知自动关闭
- **手动关闭**：支持手动关闭通知
