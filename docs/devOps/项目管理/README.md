## 📋 概述

温柔陪伴助手（YiPet）项目管理目录包含完整的项目管理体系，采用最新的任务看板管理方式，实现任务的可视化管理和实时追踪。项目管理与devOps并行运作，共同保障项目的顺利交付和持续运行。

### 🎯 管理体系特点

- 📊 **任务看板管理** - 通过可视化看板管理任务，状态清晰可见，支持实时更新和团队协作。
- 🚀 **敏捷开发方法** - 采用2周Sprint迭代，快速交付可用功能，每日站会及时调整计划。
- ⏰ **80/20时间分配** - 80%时间专注于业务价值交付，20%时间处理技术债务，提高代码质量。
- 📝 **完整对齐机制** - 与核心功能、需求文档、实施计划、规格文档、测试报告完全对齐，确保项目一致性。

## 📋 使用指南

### 1. 项目启动
1. 查看 [项目路线图](./项目路线图.md) 了解项目阶段划分和关键里程碑
2. 参考 [项目路线图](./项目路线图.md) 了解详细时间表

### 2. 日常管理
1. 在对应的任务板上查看任务状态，了解项目整体进度
2. 使用 [任务管理](./任务管理.md) 掌握任务流程和状态转换规则
3. 参考 [任务管理](./任务管理.md) 跟踪项目进度和风险
4. 查看 [资源管理](./资源管理.md) 了解团队成员分工和资源配置

### 3. 协作机制
1. 每日站会，快速同步项目状态和解决问题

### 4. 风险评估
1. 查看 [迭代任务](./任务板/迭代任务.md) 了解任务状态分布和问题与风险分析
2. 参考 [项目路线图](./项目路线图.md) 了解阶段风险评估和应对措施
3. 使用 [任务管理](./任务管理.md) 跟踪风险状态和缓解措施
