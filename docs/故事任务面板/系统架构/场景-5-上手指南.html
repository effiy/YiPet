YiPet Developer Onboarding Guide

Learning path, setup flow, debug entry points, and code patterns for new developers

First 30 Minutes

  • • Read CLAUDE.md for project overview and constraints
  • • Read core/config.js for all endpoint definitions
  • • Skim PetManager.core.js — the 800+ line orchestration hub
  • • Open any page to see the pet in action

Debug Quick Reference

  • • window.__YIPET__ — global debug object in console
  • • chrome.storage.local — inspect state in DevTools
  • • Service Worker console — background script logs
  • • Refresh extension after code changes (chrome://extensions)

Coding Conventions

  • • IIFE modules: no build tool, no import/export
  • • Vue 3 Options API for all new components
  • • Always precheck chrome API availability
  • • API endpoints in core/config.js ENDPOINTS map
