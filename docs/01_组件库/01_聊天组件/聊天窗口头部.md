# ChatHeader 组件

> 温柔陪伴助手 - 聊天窗口头部组件

---

## 概述

ChatHeader 是温柔陪伴助手的聊天窗口头部组件，提供窗口标题和控制按钮。

---

## 文件位置

```
modules/pet/components/chat/ChatHeader/
├── index.html         # 组件模板
└── index.js           # 组件逻辑
```

---

## 功能特性

- **窗口标题**：显示聊天窗口标题
- **关闭按钮**：提供窗口关闭功能
- **最小化按钮**：支持窗口最小化
