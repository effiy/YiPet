# 键盘快捷键

> 快捷操作，提高效率

---

## 功能概述

键盘快捷键功能允许用户通过键盘组合键快速执行常用操作，无需使用鼠标点击，提高工作效率。

---

## 默认快捷键

### 1. 切换宠物显示/隐藏
- **Windows/Linux**: `Ctrl + Shift + P`
- **Mac**: `Cmd + Shift + P`
- **功能**: 快速切换宠物在网页上的显示和隐藏状态
- **适用场景**: 临时隐藏宠物以获得更大的屏幕空间

### 2. 打开/关闭聊天窗口
- **Windows/Linux**: `Ctrl + Shift + X`
- **Mac**: `Cmd + Shift + X`
- **功能**: 快速打开或关闭聊天窗口
- **适用场景**: 快速启动对话，无需双击宠物

---

## 使用说明

### 使用快捷键
1. 确保扩展已正确加载
2. 在任意网页上按下快捷键组合
3. 相应功能立即执行

### 冲突处理
如果快捷键与其他扩展或浏览器快捷键冲突：
- 尝试修改其他扩展的快捷键
- 或在扩展设置中修改本扩展的快捷键

## 技术实现

### 相关文件
- `manifest.json` - 快捷键定义
- `modules/extension/background/index.js` - 快捷键事件处理
- `modules/pet/content/petManager.events.js` - 快捷键响应

### 配置格式
在 `manifest.json` 中配置：
```json
{
  "commands": {
    "toggle-pet": {
      "suggested_key": {
        "default": "Ctrl+Shift+P",
        "mac": "Command+Shift+P"
      },
      "description": "切换宠物显示"
    },
    "open-chat": {
      "suggested_key": {
        "default": "Ctrl+Shift+X",
        "mac": "Command+Shift+X"
      },
      "description": "打开聊天窗口"
    }
  }
}
```

### 事件处理
```javascript
// 后台脚本处理
chrome.commands.onCommand.addListener((command) => {
  if (command === 'toggle-pet') {
    // 切换宠物显示
  } else if (command === 'open-chat') {
    // 打开聊天窗口
  }
});
```

## 自定义快捷键

### 修改快捷键
1. 打开 Chrome 扩展管理页面 (`chrome://extensions/`)
2. 点击左上角的菜单按钮
3. 选择"键盘快捷键"
4. 找到"温柔陪伴助手"扩展
5. 点击快捷键输入框，按下新的组合键

### 添加新快捷键
开发者可以在 `manifest.json` 中添加新的快捷键：
```json
{
  "commands": {
    "take-screenshot": {
      "suggested_key": {
        "default": "Ctrl+Shift+S",
        "mac": "Command+Shift+S"
      },
      "description": "启动截图"
    }
  }
}
```

## 快捷键规范

### 组合键原则
- 使用 `Ctrl` (Windows/Linux) 或 `Cmd` (Mac) 作为主要修饰键
- 配合 `Shift` 键避免与浏览器默认快捷键冲突
- 选择易于记忆的字母（如 P = Pet, X = 聊天窗口）

### 避免冲突
- 避免使用 `Ctrl+C`、`Ctrl+V` 等常用快捷键
- 避免使用 `Ctrl+T`、`Ctrl+W` 等浏览器快捷键
- 测试快捷键在不同操作系统下的兼容性
