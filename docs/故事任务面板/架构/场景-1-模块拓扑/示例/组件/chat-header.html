ChatHeader 使用事例

概述

ChatHeader 渲染聊天窗口顶部栏，显示当前会话标题，提供认证/外部链接/关闭/侧边栏切换等操作按钮。支持模板和纯 DOM 两种模式。

组件接口

loadTemplate + createComponent

export {
  loadTemplate,    // 异步加载 HTML 模板 (#yi-pet-chat-header-template)
  createComponent   // 创建 Vue defineComponent
}
