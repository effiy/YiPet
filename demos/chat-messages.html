ChatMessages 使用事例

概述

ChatMessages 管理消息列表的渲染。支持四种视图状态 (loading/error/empty/messages)、用户消息与宠物消息差异化渲染、流式输出动画、消息操作按钮注入、Mermaid 图表渲染。

视图状态管理

四种状态 · 通过 instance._setMessagesViewState 切换

// ChatWindow 通过以下方法控制 ChatMessages 状态
inst._setMessagesViewState = (state, payload) => {
  viewState.value = state          // 'loading' | 'error' | 'empty' | 'messages'
  viewStatePayload.value = payload // 附加数据
}

// 消息列表操作
inst._messagesSet(list)              // 全量设置消息列表
inst._messagesAppend(msg)            // 追加一条消息
inst._messagesUpdateContent(idx, content)  // 更新指定消息内容（流式）
inst._messagesSetStreaming(idx, streaming) // 设置流式状态
inst._messagesClear()                // 清空所有消息
inst._getMessagesList() → Array      // 获取当前消息列表

消息类型与渲染

消息对象结构

const userMsg = {
  type: 'user',           // 'user' | 'pet'
  content: '你好',        // 消息正文（支持 Markdown HTML）
  timestamp: 1717500000000,  // Unix 毫秒时间戳
  imageDataUrl: null,     // 可选：图片 dataURL
  streaming: false,       // 是否流式输出中
  error: false,           // 是否错误消息
  aborted: false          // 是否已中止
}

// CSS class 自动计算: is-user | is-pet | is-streaming | is-error | is-aborted
messageClass(msg) → string
