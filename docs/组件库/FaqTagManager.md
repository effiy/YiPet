# FaqTagManager 组件

> FAQ 标签管理器组件，管理 FAQ 标签分类

---

## 📋 组件概述

FaqTagManager 组件负责管理 FAQ 标签，包括创建、编辑、删除和颜色设置。

---

## 🏗️ 文件结构

```
modules/pet/components/manager/FaqTagManager/
├── index.html                 # 组件 HTML 模板
└── index.js                   # Vue 应用逻辑
```

---

## 🔧 主要功能

### 1. 标签管理
- 创建新标签
- 编辑标签名称
- 删除标签
- 标签重排

### 2. 标签样式
- 标签颜色设置
- 标签图标选择
- 自定义标签背景

### 3. 标签应用
- 为 FAQ 分配标签
- 批量标签管理
- 标签使用统计
