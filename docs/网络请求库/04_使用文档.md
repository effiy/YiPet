## RequestClient 使用指南

### 创建实例

```javascript
// 使用默认配置
const client = new RequestClient();

// 自定义配置
const client = new RequestClient({
  timeout: 60000,              // 超时时间（毫秒），默认 30000
  mode: 'cors',                // 请求模式，默认 'cors'
  credentials: 'include',      // 凭证模式，默认 'omit'
  headers: {                   // 默认请求头
    'Content-Type': 'application/json',
    'X-Custom-Header': 'value'
  }
});

// 使用工厂函数
const client = createRequestClient({ timeout: 60000 });
```

### 发送请求

#### GET 请求

```javascript
// 简单 GET
const data = await client.get('https://api.example.com/items');

// 带查询参数
const data = await client.get('https://api.example.com/items', {
  page: 1,
  limit: 20,
  sort: '-createdAt'
});

// 完整配置
const data = await client.get('https://api.example.com/items', { page: 1 }, {
  headers: { 'X-Custom': 'value' },
  abortKey: 'get-items'  // 可用于取消请求
});
```

#### POST 请求

```javascript
// JSON 数据
const result = await client.post('https://api.example.com/items', {
  name: 'Test',
  description: 'Description'
});

// FormData
const formData = new FormData();
formData.append('file', file);
const result = await client.post('https://api.example.com/upload', formData);

// 文本数据
const result = await client.post('https://api.example.com/text', 'plain text');
```

#### PUT 请求

```javascript
const result = await client.put('https://api.example.com/items/1', {
  name: 'Updated Name'
});
```

#### DELETE 请求

```javascript
const result = await client.delete('https://api.example.com/items/1');
```

#### 通用 request 方法

```javascript
const result = await client.request({
  url: 'https://api.example.com/items',
  method: 'PATCH',  // GET/POST/PUT/DELETE/PATCH 等
  params: { ... },  // 查询参数
  data: { ... },    // 请求体数据
  headers: { ... }, // 自定义请求头
  timeout: 30000,   // 超时时间
  abortKey: 'my-request'  // 取消标识
});
```

### 取消请求

```javascript
// 发送请求时指定 abortKey
client.post('https://api.example.com/large-request', data, {
  abortKey: 'upload-1'
});

// 之后可以取消
client.abort('upload-1');

// 销毁客户端，取消所有请求
client.destroy();
```

### 重试机制

RequestClient 自动重试以下情况：
- 网络错误（Failed to fetch）
- 超时错误（AbortError）

**重试策略**：
- 最多重试 3 次
- 延迟时间指数增长：1s → 2s → 4s
- 不需要手动配置，自动生效

## TokenManager 使用指南

### 基本操作

```javascript
// 获取 Token（异步，优先 Chrome 存储）
const token = await tokenManager.getToken();
console.log('Token:', token);

// 获取 Token（同步，优先内存缓存）
const token = tokenManager.getTokenSync();
console.log('Token:', token);

// 保存 Token
const success = await tokenManager.saveToken('my-api-token-123');
console.log('保存成功:', success);

// 清除 Token
await tokenManager.clearToken();

// 检查是否有 Token（异步）
const hasToken = await tokenManager.hasToken();

// 检查是否有 Token（同步）
const hasToken = tokenManager.hasTokenSync();
```

### 环境变量注入

TokenManager 会按优先级检查以下环境变量：

```javascript
// 1. Content Script 环境（最高优先级）
window.__API_X_TOKEN__ = 'my-token';

// 2. Node.js 环境
process.env.API_X_TOKEN = 'my-token';

// 3. Service Worker 环境
self.__API_X_TOKEN = 'my-token';
```

如果设置了环境变量，TokenManager 会优先使用，不读取 Chrome 存储。

### 验证 Token 格式

```javascript
const isValid = tokenManager.validateToken('my-token-123');
console.log('Token 有效:', isValid);
```

验证规则：
- 非空且为字符串
- 长度 ≥ 10 个字符
- 只包含字母、数字、下划线、连字符

### 使用 TokenUtils 工具

```javascript
// 等价于 tokenManager 的便捷接口
const token = await TokenUtils.getApiToken();
const tokenSync = TokenUtils.getApiTokenSync();
await TokenUtils.saveApiToken('token');
const hasToken = await TokenUtils.hasApiToken();
const hasTokenSync = TokenUtils.hasApiTokenSync();
```

## ErrorHandler 使用指南

### 错误类型

网络请求库定义了 6 种错误类型：

```javascript
// 基础错误类
throw new APIError('错误信息', 'ERROR_CODE', { details: '详细信息' });

// 网络错误
throw new NetworkError('网络连接失败', { originalError: err });

// 超时错误
throw new TimeoutError('请求超时', 30000, { originalError: err });

// 认证错误
throw new AuthError('认证失败', { originalError: err });

// 验证错误
throw new ValidationError('数据验证失败', { name: '必填' }, { originalError: err });

// 限流错误
throw new RateLimitError('请求过于频繁', 60, { originalError: err });
```

### 使用 ErrorHandler

```javascript
const errorHandler = new ErrorHandler({
  maxRetries: 3,           // 最大重试次数
  retryDelay: 1000,        // 初始重试延迟（毫秒）
  onError: async (error, context) => {
    // 自定义错误处理回调
    console.error('发生错误:', error, context);
    await reportToMonitoring(error);
  }
});

// 处理错误
try {
  await makeRequest();
} catch (err) {
  const handledError = await errorHandler.handle(err, {
    endpoint: '/api/items',
    retryCount: 0
  });
  // handledError 是分类后的 APIError 子类实例
}
```

### 使用便捷函数

```javascript
// 创建错误
const err = createError('出错了', 'MY_CODE', { info: 'details' });

// 格式化错误
const formatted = formatError(err);
console.log(formatted);
// {
//   name: 'APIError',
//   message: '出错了',
//   code: 'MY_CODE',
//   details: { info: 'details' },
//   timestamp: 1234567890
// }

// 设置全局错误处理器
setGlobalErrorHandler(customHandler);

// 获取全局错误处理器
const handler = getGlobalErrorHandler();
```

### 错误处理建议

```javascript
try {
  const data = await client.get('/api/items');
  return data;
} catch (err) {
  if (err instanceof NetworkError) {
    // 网络错误：提示用户检查网络
    showToast('网络连接失败，请检查网络后重试');
  } else if (err instanceof TimeoutError) {
    // 超时错误：提示用户稍后重试
    showToast('请求超时，请稍后重试');
  } else if (err instanceof AuthError) {
    // 认证错误：引导用户重新登录
    redirectToLogin();
  } else if (err instanceof RateLimitError) {
    // 限流错误：提示用户等待
    showToast(`请求过于频繁，请等待 ${err.retryAfter} 秒后重试`);
  } else {
    // 其他错误：通用提示
    showToast('操作失败，请稍后重试');
  }
}
```

## Logger 使用指南

### 基本使用

```javascript
// 使用全局单例
logger.info('应用启动');
logger.warn('配置未找到，使用默认值');
logger.error('发生错误', error);

// 创建自定义实例
const myLogger = new Logger({
  level: LOG_LEVELS.DEBUG,  // 日志级别
  prefix: '[MyModule]',     // 日志前缀
  enabled: true,            // 是否启用
  includeTimestamp: true    // 是否包含时间戳
});

// 使用工厂函数
const logger = createLogger({ prefix: '[MyModule]' });
```

### 日志级别

```javascript
// 级别从低到高
LOG_LEVELS.DEBUG   // 0 - 调试信息
LOG_LEVELS.INFO    // 1 - 一般信息（默认）
LOG_LEVELS.WARN    // 2 - 警告信息
LOG_LEVELS.ERROR   // 3 - 错误信息
LOG_LEVELS.NONE    // 4 - 关闭日志

// 设置级别
logger.setLevel(LOG_LEVELS.DEBUG);  // 显示 DEBUG 及以上
logger.setLevel(LOG_LEVELS.WARN);   // 只显示 WARN 及 ERROR

// 获取当前级别
const currentLevel = logger.getLevel();
```

### 日志方法

```javascript
// 一般日志
logger.debug('调试信息', data);
logger.info('一般信息', data);
logger.warn('警告信息', data);
logger.error('错误信息', error);

// 专用日志（自动格式化）
logger.logRequest({
  url: 'https://api.example.com/items',
  method: 'GET',
  params: { page: 1 }
});

logger.logResponse({ data: '响应数据' });

logger.logError(error, { endpoint: '/api/items' });
```

### 启用/禁用

```javascript
logger.disable();  // 禁用日志输出
logger.enable();   // 重新启用
```

### 使用 LoggerUtils

```javascript
// 等价于全局 logger 的便捷接口
LoggerUtils.logRequest(config);
LoggerUtils.logResponse(response);
LoggerUtils.logError(error, context);
LoggerUtils.setLogLevel(LOG_LEVELS.DEBUG);
LoggerUtils.getLogLevel();
```

## ApiManager 使用指南

### 创建服务实例

```javascript
// 基础用法
const api = new ApiManager('https://api.example.com');

// 完整配置
const api = new ApiManager('https://api.example.com', {
  enabled: true,                    // 是否启用
  timeout: 30000,                   // 超时时间
  clientOptions: { ... },           // RequestClient 配置
  errorHandler: { ... },            // ErrorHandler 配置
  logger: {                         // Logger 配置
    prefix: '[MyService]',
    level: LOG_LEVELS.INFO
  }
});
```

### 发送请求

```javascript
// 便捷方法
const data = await api.get('/items', { page: 1 });
const result = await api.post('/items', { name: 'Test' });
const result = await api.put('/items/1', { name: 'Updated' });
const result = await api.delete('/items/1');

// 通用 request 方法
const result = await api.request('/items', {
  method: 'POST',
  data: { ... },
  params: { ... },
  headers: { ... }
});
```

### 使用拦截器

```javascript
// 添加请求拦截器
api.addRequestInterceptor(async (config) => {
  // 修改请求配置
  config.headers['X-Request-Id'] = generateId();
  console.log('发送请求:', config.url);
  return config;  // 必须返回修改后的配置
});

// 添加响应拦截器
api.addResponseInterceptor(async (response) => {
  // 修改响应数据
  console.log('收到响应:', response);
  return transformResponse(response);  // 必须返回处理后的响应
});

// 可以添加多个拦截器，按添加顺序执行
api.addRequestInterceptor(interceptor1);
api.addRequestInterceptor(interceptor2);  // 在 interceptor1 之后执行
```

### 默认拦截器

ApiManager 自动设置两个拦截器：

1. **Token 拦截器**: 自动从 TokenManager 获取 Token，添加到请求头 `X-Token`
2. **日志拦截器**: 自动记录请求和响应信息

不需要手动配置，开箱即用。

## 常见问题

### Q: Token 如何设置？

有三种方式：
1. **环境变量**: 设置 `window.__API_X_TOKEN__`（优先级最高）
2. **Chrome 存储**: 调用 `tokenManager.saveToken('token')`
3. **手动添加**: 直接在请求拦截器中添加 `config.headers['X-Token'] = 'token'`

### Q: 如何自定义请求头？

```javascript
// 方法 1: 创建时设置默认头
const client = new RequestClient({
  headers: { 'X-Custom': 'value' }
});

// 方法 2: 单次请求时设置
await client.get('/url', params, {
  headers: { 'X-Custom': 'value' }
});

// 方法 3: 使用拦截器
api.addRequestInterceptor((config) => {
  config.headers['X-Custom'] = 'value';
  return config;
});
```

### Q: 如何处理跨域问题？

RequestClient 默认 `mode: 'cors'`，如果后端支持 CORS 则不需要额外配置。如果遇到跨域问题：
1. 检查后端 Access-Control-Allow-Origin 头
2. 检查后端 Access-Control-Allow-Methods 头
3. 检查后端 Access-Control-Allow-Headers 头

### Q: 如何调试请求？

```javascript
// 1. 降低日志级别
logger.setLevel(LOG_LEVELS.DEBUG);

// 2. 查看请求/响应日志
// Logger 会自动记录所有请求和响应

// 3. 查看统计信息
console.log(api.getStats());
```

### Q: 队列保存失败会重试吗？

会的。SessionService 的 `_processSaveQueue` 会自动重试失败的会话，最多 3 次。重试时会重新加入队列等待下一批处理。

### Q: 如何取消一个队列中的会话保存？

目前不支持单独取消队列中的某项。如果需要取消，建议：
1. 等待队列自动处理完成
2. 或者调用 `flushSaveQueue()` 立即处理

## 最佳实践

### 1. 单例复用

```javascript
// ✅ 推荐：应用内复用一个实例
const api = new ApiManager(baseUrl);
// 在各模块中共享使用

// ❌ 不推荐：每次请求创建新实例
const api = new ApiManager(baseUrl);  // 浪费资源
await api.get('/url');
```

### 2. 错误处理

```javascript
// ✅ 推荐：分类处理不同错误
try {
  await api.get('/url');
} catch (err) {
  if (err instanceof NetworkError) {
    showNetworkError();
  } else if (err instanceof AuthError) {
    redirectToLogin();
  } else {
    showGenericError();
  }
}

// ❌ 不推荐：不处理或笼统处理
try {
  await api.get('/url');
} catch (err) {
  console.error(err);  // 用户看不到反馈
}
```
