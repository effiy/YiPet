# SessionInfoEditor 组件

> 温柔陪伴助手 - 会话信息编辑器组件

---

## 概述

SessionInfoEditor 是温柔陪伴助手的会话信息编辑器组件，用于编辑会话的基本信息。

---

## 文件位置

```
modules/pet/components/editor/SessionInfoEditor/
├── index.html         # 组件模板
└── index.js           # 组件逻辑
```

---

## 功能特性

- **会话名称编辑**：支持编辑会话名称
- **会话描述编辑**：支持编辑会话描述
- **保存功能**：支持保存会话信息
