# SessionTagManager 组件

> 温柔陪伴助手 - 会话标签管理器组件

---

## 概述

SessionTagManager 是温柔陪伴助手的会话标签管理器组件，用于管理会话的标签。

---

## 文件位置

```
modules/pet/components/manager/SessionTagManager/
├── index.html         # 组件模板
└── index.js           # 组件逻辑
```

---

## 功能特性

- **标签列表**：显示标签列表
- **添加标签**：支持添加新标签
- **删除标签**：支持删除标签
- **标签分配**：支持为会话分配标签
