# 管理对话会话

## 功能说明

保存和管理多个对话会话，支持标签分类。

## 功能分类

📦 数据管理

## 使用方法

- **创建会话**：会话列表 → 新建会话
- **切换会话**：直接点击会话
- **编辑会话**：点击编辑按钮

## 相关文件

- `modules/pet/content/petManager.session.js`
- `modules/session/` (会话功能模块)
- `core/api/services/SessionService.js`

## 功能特点

- 创建新会话
- 切换不同会话
- 编辑会话信息
- 会话标签分类
- 会话搜索功能
- 云端同步（可选）
