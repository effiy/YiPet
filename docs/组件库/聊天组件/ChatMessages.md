# ChatMessages 组件

> 温柔陪伴助手 - 聊天消息列表组件

---

## 概述

ChatMessages 是温柔陪伴助手的聊天消息列表组件，用于显示聊天消息列表。

---

## 文件位置

```
modules/pet/components/chat/ChatMessages/
├── index.html         # 组件模板
└── index.js           # 组件逻辑
```

---

## 功能特性

- **消息列表**：显示聊天消息列表
- **自动滚动**：支持消息自动滚动
- **加载状态**：显示消息加载状态
