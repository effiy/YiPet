jsdom 使用事例

概述

jsdom v26.0.0 为 vitest 提供浏览器环境。无需真实 Chrome 即可运行全部测试 —— 提供 window/document/navigator/AbortController/DOM 事件模拟。

全局环境注入

vitest 配置 environment: 'jsdom' 后，所有测试可访问 DOM API。

setup.mjs 中的环境检测

import { beforeAll } from 'vitest';

beforeAll(() => {
  if (typeof window === 'undefined') {
    throw new Error('jsdom not configured');
  }
  // 确认环境就绪
});

可用 API 清单

jsdom 提供的浏览器 API

window → [object Window]
document → [object Document]
navigator → [object Navigator]
AbortController → [object AbortController]
HTMLElement → [object HTMLElement]
Event/CustomEvent → [object Function]
localStorage → [object Storage]

Vue 3 组件挂载

jsdom 环境下通过 test-utils 模拟 Vue 3 组件挂载。

// 模拟 Vue 组件挂载
const mountEl = document.createElement('div');
mountEl.id = 'app';
document.body.appendChild(mountEl);

const app = Vue.createApp({
  template: '<ChatWindow />',
  components: { ChatWindow }
});
app.mount(mountEl);

expect(mountEl.querySelector('.chat-window')).not.toBeNull();
