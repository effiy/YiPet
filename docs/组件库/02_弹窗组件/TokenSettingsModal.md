# TokenSettingsModal 组件

> 温柔陪伴助手 - Token 设置弹窗组件

---

## 概述

TokenSettingsModal 是温柔陪伴助手的 Token 设置弹窗组件，用于配置 API Token。

---

## 文件位置

```
modules/pet/components/modal/TokenSettingsModal/
├── index.html         # 组件模板
└── index.js           # 组件逻辑
```

---

## 功能特性

- **Token 输入**：提供 Token 输入框
- **保存 Token**：支持保存 Token
- **删除 Token**：支持删除 Token
