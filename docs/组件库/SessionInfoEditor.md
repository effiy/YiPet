# SessionInfoEditor 组件

> 会话信息编辑器组件，编辑会话的详细信息

---

## 📋 组件概述

SessionInfoEditor 组件允许用户编辑会话的详细信息，包括会话名称、描述、标签等。

---

## 🏗️ 文件结构

```
modules/pet/components/editor/SessionInfoEditor/
├── index.html                 # 组件 HTML 模板
└── index.js                   # Vue 应用逻辑
```

---

## 🔧 主要功能

### 1. 会话信息编辑
- 编辑会话名称
- 编辑会话描述
- 添加和删除标签
- 标记会话为收藏

### 2. 历史记录
- 显示会话创建时间
- 显示会话最后更新时间
- 显示消息数量统计

### 3. 操作功能
- 保存会话信息
- 取消编辑
- 会话删除功能
