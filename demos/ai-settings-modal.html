AiSettingsModal 使用事例

概述

AiSettingsModal 是 AI 模型选择弹窗，极简设计仅 107 行代码。提供模型名称输入、Token 设置入口、取消/保存操作。支持模板和 h() 双模式渲染。

状态管理

const store = reactive({
  selectedModel: '',   // 当前输入的模型名称
})

// 生命周期: 打开弹窗时从 manager.currentModel 初始化
store.selectedModel = manager.currentModel || ''

// 三个方法
openToken()   // manager.openAuth() — 打开 Token 设置弹窗
cancel()      // 关闭弹窗，恢复原模型值
save()        // manager.currentModel = selectedModel → saveState → close

渲染模式

双模式：模板 + h() 降级

export function openAiSettingsModal(manager) {
  const store = reactive({ selectedModel: manager.currentModel || '' })

  const component = {
    template: '#yi-pet-ai-settings-template',  // 模板模式
    // 或 render() { return h('div', ...) }          // 降级模式
    methods: { openToken, cancel, save }
  }

  const app = createApp(component)
  app.mount(container)
}
