# 安全规范

> 安全编码规范和最佳实践

---

## 📋 概述

本文档定义了温柔陪伴助手开发过程中的安全编码规范，确保代码的安全性和用户数据的保护。

---

## 🔒 数据安全

### 1. 用户数据保护
- API 令牌和敏感信息必须加密存储在 Chrome `storage.local` 中
- 不能在控制台输出敏感信息（如 API Token）
- 用户数据不能发送到非信任的第三方服务器
- 所有用户输入必须进行验证和过滤

### 2. 数据传输安全
- API 通信必须使用 HTTPS
- 验证服务器证书的有效性
- 避免在 URL 中传递敏感信息
- 使用 POST 方法发送敏感数据

### 3. 数据存储安全
```javascript
// ✅ 正确做法：使用 Chrome 安全存储
chrome.storage.local.set({
  petSettings: {
    apiToken: 'encrypted-token-here'
  }
});

// ❌ 错误做法：不要使用 localStorage
localStorage.setItem('apiToken', 'plain-text-token');
```

## 💉 输入验证

### 1. 用户输入验证
- 所有用户输入必须进行验证
- 使用白名单验证而非黑名单
- 限制输入长度和格式
- 对特殊字符进行转义处理

### 2. 消息内容过滤
- 防止 XSS 攻击
- 对渲染的 HTML 进行严格过滤
- 使用 DOMPurify 等库进行 HTML 净化
- 避免直接使用 innerHTML

### 3. 代码示例
```javascript
// ✅ 正确做法：输入验证
function sanitizeInput(input) {
  const maxLength = 4000;
  if (input.length > maxLength) {
    throw new Error('Input exceeds maximum length');
  }
  return input.trim();
}

// ❌ 错误做法：不验证直接使用
function processInput(input) {
  return input; // 可能包含恶意代码
}
```

## 🔐 权限管理

### 1. 扩展权限
- 只请求必要的权限
- 避免使用 `all_urls` 权限，除非确实需要
- 使用 `activeTab` 权限替代 `<all_urls>`
- 定期审查权限需求

### 2. Manifest 配置
```json
{
  "permissions": [
    "storage",
    "activeTab",
    "tabs"
  ],
  "host_permissions": [
    "https://api.effiy.cn/*"
  ]
}
```

## 🛡️ XSS 防护

### 1. Markdown 渲染安全
- Markdown 渲染前进行净化
- 只允许安全的 HTML 标签
- 禁用脚本执行
- 使用 `marked` 库时启用安全选项

### 2. 示例代码
```javascript
// ✅ 正确做法：安全渲染
marked.setOptions({
  sanitize: true,
  mangle: true,
  headerIds: false
});

const safeHtml = marked.parse(markdown);
```

## 🐛 错误处理安全

### 1. 错误信息处理
- 不要在错误消息中暴露敏感信息
- 使用通用错误消息面向用户
- 详细错误信息只在开发模式显示

### 2. 日志安全
- 日志中不要包含敏感数据
- 避免记录完整的用户输入
- 定期审查日志内容

---

## 🔄 依赖安全

### 1. 第三方库
- 使用稳定版本的第三方库
- 定期检查和更新依赖
- 验证库的来源和可信度
- 避免使用不再维护的库

### 2. 库使用规范
```javascript
// ✅ 正确做法：使用官方 CDN 或本地文件
<script src="libs/vue.global.js"></script>

// ❌ 错误做法：不要使用不可信的 CDN
<script src="https://untrusted-cdn.com/vue.js"></script>
```

---

## 🔍 安全审查

### 1. 代码审查
- 所有代码提交必须经过安全审查
- 重点审查用户输入处理
- 检查权限请求的合理性

### 2. 安全测试
- 定期进行安全测试
- 检查常见漏洞
- 使用安全扫描工具
