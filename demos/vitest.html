vitest 使用事例

概述

vitest 是项目测试框架的核心运行器，提供断言、Mock、时间控制等能力。13 个测试文件、250 个用例全部基于 vitest。

核心 API — describe / it / expect

所有 250 个用例均使用此三元组模式：

基础断言模式

// tests/unit/config.test.mjs
import { describe, it, expect, beforeEach } from 'vitest';

describe('Config', () => {
  beforeEach(async () => {
    delete globalThis.PET_CONFIG;
    await loadModule('core/config.js');
  });

  it('should have DEFAULT_CONFIG with required keys', () => {
    expect(globalThis.PET_CONFIG).toBeDefined();
    expect(globalThis.PET_CONFIG.DEFAULT_CONFIG.apiBaseUrl)
      .toBe('https://api.effiy.cn');
  });
});

Mock 模式

vi.fn() — 函数 Mock 三类模式

// 模式 1: 替换全局函数
globalThis.fetch = vi.fn();

// 模式 2: 替换实例方法加速异步等待
globalThis.RequestClient.prototype._delay = () => Promise.resolve();

// 模式 3: 记录调用参数
const [url, init] = globalThis.fetch.mock.calls[0];
expect(init.method).toBe('GET');

mockResolvedValue / mockRejectedValue — 返回值控制

// 单次成功
globalThis.fetch.mockResolvedValueOnce(mockJsonResponse({ data: 'hello' }));

// 先失败后成功（测试重试）
globalThis.fetch
  .mockRejectedValueOnce(new TypeError('Failed to fetch'))
  .mockResolvedValueOnce(mockJsonResponse({ data: 'retry-ok' }));

// 永不 resolve（测试超时）
globalThis.fetch.mockReturnValue(new Promise(() => {}));

时间控制

vi.useFakeTimers + vi.advanceTimersByTimeAsync

it('rejects with timeout error', async () => {
  vi.useFakeTimers();
  globalThis.fetch.mockReturnValue(new Promise(() => {}));

  const client = new RequestClient({ timeout: 100 });
  const prom = client.get('https://api.example.com/slow');

  await vi.advanceTimersByTimeAsync(150);
  await expect(prom).rejects.toThrow(/超时/);

  vi.useRealTimers();
});

配置文件

import { defineConfig } from 'vitest/config';

export default defineConfig({
  test: {
    // jsdom 浏览器环境模拟
    environment: 'jsdom',
    // 全局 describe/it/expect
    globals: true,
    // 启动前执行
    setupFiles: ['./tests/setup.mjs'],
    include: ['tests/**/*.test.mjs'],
    coverage: {
      provider: 'v8',
      include: ['core/**/*.js', 'modules/**/*.js'],
      reporter: ['text', 'json', 'html'],
    },
  },
});
