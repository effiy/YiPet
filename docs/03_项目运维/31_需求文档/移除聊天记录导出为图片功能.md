# 需求文档：移除聊天记录导出为图片功能

## 1. 需求概述

**功能名称**：移除聊天记录导出为图片功能

**需求类型**：功能移除

**优先级**：中等

**背景**：当前应用支持将聊天记录导出为高清 PNG 图片的功能，但由于该功能存在性能问题、依赖外部库（html2canvas）以及用户使用频率较低等原因，决定将该功能从应用中移除。

## 2. 功能分析

### 当前功能实现

#### 2.1 文件结构
```
├── modules/chat/content/export-chat-to-png.js     # 导出聊天记录为图片的核心实现
├── modules/pet/content/core/petManager.core.js    # 添加导出按钮到消息的方法
├── libs/html2canvas.min.js                         # 依赖的截图库（340KB）
```

#### 2.2 功能特点
- **导出聊天记录为 PNG 图片**：支持导出整个聊天会话或单条消息
- **高清导出**：使用 2x 缩放，生成 1500px 宽度的高清图片，适合手机查看
- **样式处理**：复制计算样式以确保导出图片的视觉一致性
- **图片加载**：等待所有图片加载完成，包括超时保护
- **错误处理**：完善的加载提示和错误处理机制

#### 2.3 导出流程
1. 在每条消息的操作按钮中添加"导出消息为图片"按钮
2. 点击按钮后显示加载提示
3. 使用 html2canvas 库将 DOM 元素转换为 canvas
4. 转换为 PNG 图片并下载
5. 显示成功/失败提示

## 3. 移除范围

### 3.1 需要移除的文件

1. **`modules/chat/content/export-chat-to-png.js`** - 导出功能核心实现文件
2. **`libs/html2canvas.min.js`** - 依赖的截图库
3. **在 `manifest.json` 中的相关配置** - web_accessible_resources 中的引用

### 3.2 需要修改的文件

1. **`modules/pet/content/core/petManager.core.js`** - 移除 `addExportButtonForMessage` 方法
2. **`modules/pet/components/chat/ChatWindow/index.js`** - 移除导出功能相关的导入和调用
3. **`CLAUDE.md`** - 更新文档，移除导出功能的描述

## 4. 实施步骤

### 步骤 1：删除导出功能实现文件
```bash
rm -f modules/chat/content/export-chat-to-png.js
rm -f libs/html2canvas.min.js
```

### 步骤 2：移除按钮添加方法
在 `modules/pet/content/core/petManager.core.js` 中：
- 删除 `addExportButtonForMessage` 方法（第 791-833 行）
- 移除在 `addActionButtonsToMessage` 方法中对该方法的调用

### 步骤 3：更新 ChatWindow 组件
在 `modules/pet/components/chat/ChatWindow/index.js` 中：
- 移除对 `export-chat-to-png.js` 的导入
- 移除相关的导出功能调用

### 步骤 4：更新 manifest.json
在 `manifest.json` 中：
- 移除 `web_accessible_resources` 中对 `html2canvas.min.js` 和 `export-chat-to-png.js` 的引用

### 步骤 5：更新文档
在 `CLAUDE.md` 中：
- 移除"导出聊天记录为图片"功能的描述
- 更新技术栈部分，移除 html2canvas 库的说明

### 步骤 6：清理相关代码
- 检查是否还有其他文件引用了导出功能
- 移除相关的 CSS 样式（如果有）

## 5. 影响分析

### 5.1 功能影响
- 移除后，用户将无法将聊天记录导出为图片
- 用户仍可使用其他导出方式（如会话导入/导出功能）
- 聊天记录的查看和管理功能保持不变

### 5.2 性能影响
- 减少了对 html2canvas 库的依赖，初始加载时间有所改善
- 移除了 DOM 样式计算和 Canvas 渲染过程，节省了内存和 CPU 资源

### 5.3 兼容性影响
- 该功能是纯前端实现，移除后不影响与后端 API 的通信
- 对现有会话数据无影响
- 保持与 Chrome 浏览器的兼容性

## 6. 替代方案

### 6.1 会话导入/导出
应用已支持将会话导出为 ZIP 文件，包含聊天记录和相关资源。用户可以：
1. 使用"导出会话"功能保存完整的会话数据
2. 导入会话到其他设备或浏览器
3. 从导出的文件中获取聊天记录的原始数据

### 6.2 复制粘贴
用户可以使用复制功能将消息内容复制到其他应用中：
1. 每条消息都有复制按钮
2. 支持 Markdown 格式的文本复制
3. 可以将内容粘贴到笔记应用或文档中
