**修复**:
- 将 `petManager.editor.js` 修正为真正的兼容层（仅保留前 11 行）

### 2.2 清理纯兼容层文件

**清理内容**:
1. 从 `manifest.json` 中移除了 `petManager.ai.js` 的引用
2. 从 `manifest.json` 中移除了 `petManager.mermaid.js` 的引用
3. 同步更新了 `injectionService.js` 中的加载列表
4. 保留了兼容层文件本身，便于回滚

**原因**:
- 这些文件仅输出日志，无实际功能
- 新模块已正常加载并运行
