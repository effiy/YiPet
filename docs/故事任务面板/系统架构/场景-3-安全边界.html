YiPet Security Boundary

认证 · Token 三级降级 · 传输加密 · 输入验证 · 存储配额 · 权限控制

Authentication

  • TokenManager: 3-level fallback (env / storage / empty)
  • X-Token injected in every API request header
  • isChromeStorageAvailable() preflight on all chrome.* calls
  • isContextInvalidated() global guard for extension reload
  • Token encrypted in chrome.storage.local, never on disk

Data Protection

  • chrome.storage.local — extension-isolated sandbox
  • Quota limit: 10MB with LRU eviction policy
  • Reconstructable data (petOssFiles) cleared first
  • StorageHelper.cleanupOldData() on quota exceeded
  • Token never persisted to source code or config files

Input & Transport

  • fetch CORS policy for all API communications
  • Content-Type constraints enforced in RequestClient
  • System page (chrome://) skip detection in content script
  • ⚠️ XSS filter: currently missing, needs hardening (P1)
  • CDN scripts loaded with SRI hash integrity checks
