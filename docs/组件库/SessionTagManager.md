# SessionTagManager 组件

> 会话标签管理器组件，管理会话标签分类

---

## 📋 组件概述

SessionTagManager 组件负责管理会话标签，包括创建、编辑、删除和管理会话的标签。

---

## 🏗️ 文件结构

```
modules/pet/components/manager/SessionTagManager/
├── index.html                 # 组件 HTML 模板
└── index.js                   # Vue 应用逻辑
```

---

## 🔧 主要功能

### 1. 标签管理
- 创建新标签
- 编辑标签名称
- 删除标签
- 标签重排

### 2. 标签应用
- 为会话分配标签
- 批量标签管理
- 标签筛选和搜索

### 3. 标签统计
- 标签使用频率
- 标签下的会话数量
- 标签使用趋势
