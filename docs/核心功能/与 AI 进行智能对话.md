# 与 AI 进行智能对话

## 功能说明

流式响应的 AI 对话体验，支持 Markdown 渲染。

## 功能分类

🐾 基础功能

## 使用方法

- **打开/关闭**：`Ctrl+Shift+X`
- **发送消息**：Enter 或点击发送按钮
- **插入截图**：点击截图按钮

## 相关文件

- `modules/pet/components/chat/ChatWindow/`
- `modules/pet/content/petManager.ai.js`
- `modules/pet/content/petManager.parser.js`

## 功能特点

- 流式响应，实时显示 AI 回复
- 支持 Markdown 格式渲染
- 代码高亮显示
- 支持截图插入
- 会话历史记录
