## §2 风险

| 风险 | 等级 | 缓解 |
|------|:---:|------|
| Content-script 注入范围过大 | 高 | 系统页面跳过检测 + manifest 精细化匹配 |
| chrome.storage.local 配额超限 | 中 | LRU 淘汰 + 可重建数据优先清理 |
| Vue 3 CDN 运行时不可控 | 低 | 预检 Vue 全局变量存在性 |
