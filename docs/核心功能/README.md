## 📋 功能列表

本目录包含温柔陪伴助手所有核心功能的详细文档：

### 🐾 基础功能
- **[虚拟宠物展示](./虚拟宠物展示.md)** - 在网页上显示可爱的虚拟宠物
- **[AI 聊天界面](./AI聊天界面.md)** - 流式响应的 AI 对话体验
- **[多种宠物角色](./多种宠物角色.md)** - 多种可爱的虚拟宠物角色可选

### 🛠️ 实用工具
- **[区域截图功能](./区域截图功能.md)** - 自由选择截图区域，便捷快速截图
- **[键盘快捷键](./键盘快捷键.md)** - 快捷操作，提高效率

### 📦 数据管理
- **[会话管理](./会话管理.md)** - 保存和管理多个对话会话
- **[FAQ 系统](./FAQ系统.md)** - 保存常用问题和答案，快速检索和复用

### 📊 增强功能
- **[Mermaid 图表渲染](./Mermaid图表渲染.md)** - 支持 Mermaid 语法的图表渲染
