# 多种宠物角色

> 多种可爱的虚拟宠物角色可选

---

## 功能概述

多种宠物角色功能提供不同类型的虚拟宠物角色，每个角色都有独特的外观和性格特点。

---

## 可用角色

### 1. 教师 📚
- **角色定位**：知识渊博的学习伙伴
- **性格特点**：耐心、智慧、乐于助人
- **适用场景**：学习辅导、知识问答、技术讨论
- **默认提示词**：你是一位知识渊博的教师角色...

### 2. 医生 👨‍⚕️
- **角色定位**：关心健康的医疗顾问
- **性格特点**：专业、关怀、温和
- **适用场景**：健康咨询、医疗建议、生活指导
- **默认提示词**：你是一位专业的医生角色...

### 3. 甜品师 👨‍🍳
- **角色定位**：甜蜜温馨的生活伴侣
- **性格特点**：温柔、甜美、乐观
- **适用场景**：日常聊天、生活建议、放松时刻
- **默认提示词**：你是一位甜蜜温馨的甜品师角色...

### 4. 警察 👮
- **角色定位**：正直可靠的安全卫士
- **性格特点**：严谨、可靠、果断
- **适用场景**：安全咨询、问题解决、决策帮助
- **默认提示词**：你是一位正直可靠的警察角色...

---

## 使用说明

### 切换角色
1. 打开聊天窗口
2. 点击设置或角色选择按钮
3. 从角色列表中选择想要的角色
4. 确认切换，宠物外观和 AI 性格会相应变化

### 自定义角色
1. 在角色设置中选择"自定义"
2. 上传角色图片或使用默认
3. 设置角色名称和性格描述
4. 配置默认提示词

## 技术实现

### 相关文件
- `modules/pet/content/modules/petManager.roles.js` - 角色配置模块
- `assets/images/` - 角色图片资源
- `modules/pet/components/modal/` - 角色选择弹窗

### 角色配置结构
```javascript
{
  id: 'teacher',
  name: '教师',
  emoji: '📚',
  description: '知识渊博的学习伙伴',
  personality: '耐心、智慧、乐于助人',
  systemPrompt: '你是一位知识渊博的教师角色...',
  imagePath: '/assets/images/教师/icon.png',
  color: '#4a90e2'
}
```

## 自定义开发

### 添加新角色

1. **准备角色图片**
   - 在 `assets/images/` 下创建新角色目录
   - 添加角色图标和动画帧

2. **配置角色信息**
   - 编辑 `petManager.roles.js`
   - 添加新角色配置对象

3. **更新清单**
   - 在 `manifest.json` 的 `web_accessible_resources` 中添加新角色资源

### 角色动画
- 支持多种动画状态（空闲、说话、行走等）
- 可通过 CSS 或 JavaScript 控制动画
- 响应角色状态变化切换动画
