## 📁 目录说明

### 顶级目录说明

| 目录/文件 | 说明 |
|-----------|------|
| `manifest.json` | Chrome 扩展清单文件，定义扩展的权限、脚本、图标等元信息 |
| `assets/` | 样式、图片、图标资源 |
| `core/` | 核心系统模块 |
| `libs/` | 第三方库 |
| `modules/` | 功能模块（按功能组织） |
| `docs/` | 文档目录 |
