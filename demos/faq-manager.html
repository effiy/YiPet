FaqManager 使用事例

概述

FaqManager 是 FAQ 常见问题管理弹窗，1000+ 行代码。支持搜索过滤、标签筛选（正向/反向/未标签）、标签管理（重命名/删除）、FAQ 增删改查、复制/发送/上下移动、键盘 Escape 关闭。

状态管理 (Store)

const store = reactive({
  allFaqs: [],              // 全部 FAQ 数据 (faq_items + 自定义)
  searchFilter: '',          // 搜索关键词
  selectedTags: [],          // 当前选中的标签
  tagFilterReverse: false,    // 反向筛选（排除已选标签）
  tagFilterNoTags: false,     // 仅显示无标签 FAQ
  visible: false,             // 弹窗可见性
  tagManagerVisible: false,    // 标签管理面板可见性
  editingTag: null,           // 正在编辑的标签
})

// 计算属性: allTags · filteredTags · visibleTags · filteredFaqs · summaryText
normalizeTags(tags) → 去重 + 小写标准化

核心方法矩阵

分类方法行为
搜索clearSearch()清空搜索框
标签筛选toggleTag(tag) · toggleReverse() · toggleNoTags() · clearTagFilters()切换标签选择/反向/无标签模式/清除全部
标签管理toggleTagManager() · renameTag(old,new) · deleteTag(tag)打开标签管理面板/重命名/删除（全局更新）
FAQ 操作onFaqInsert(q,a,tags) · onFaqSend(faq) · onFaqCopy(faq)新增/发送到聊天/复制到剪贴板
FAQ 编辑deleteFaq(index) · editTags(faq,index)删除 FAQ/打开 FaqTagManager
排序moveFaqUp(idx) · moveFaqDown(idx)上移/下移 FAQ 项
生命周期close() · refresh()关闭弹窗/刷新 FAQ 数据

渲染模式

双模式渲染：模板优先 + h() 降级

if (typeof Vue?.compile === 'function') {
  // 模式 1: 从 #yi-pet-faq-manager-template 加载 HTML 模板

} else {
  // 模式 2: 纯 render 函数 (h()) 构建完整 DOM
  return { render() { return h('div', ...) }, ... }
}
