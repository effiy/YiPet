# 使用键盘快捷键操作

## 功能说明

快捷操作，提高效率。

## 功能分类

🛠️ 实用工具

## 默认快捷键

| 功能 | Windows/Linux | Mac | 描述 |
|------|--------------|-----|------|
| 切换宠物显示/隐藏 | `Ctrl+Shift+P` | `Cmd+Shift+P` | 快速显示或隐藏宠物 |
| 打开/关闭聊天窗口 | `Ctrl+Shift+X` | `Cmd+Shift+X` | 快速打开或关闭聊天 |

## 自定义快捷键

打开 `chrome://extensions/` → 点击"键盘快捷键" → 找到"温柔陪伴助手"扩展 → 修改或添加快捷键

## 相关文件

- `manifest.json` (快捷键定义)
- `modules/extension/background/index.js` (事件处理)
