# ChatMessage 组件

> 温柔陪伴助手 - 聊天消息展示组件

---

## 概述

ChatMessage 是温柔陪伴助手的聊天消息展示组件，用于显示单个聊天消息。

---

## 文件位置

```
modules/pet/components/chat/ChatMessage/
├── index.html         # 组件模板
└── index.js           # 组件逻辑
```

---

## 功能特性

- **消息展示**：显示聊天消息内容
- **消息类型**：支持多种消息类型（文本、图片等）
- **时间显示**：显示消息发送时间
