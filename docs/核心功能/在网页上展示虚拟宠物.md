# 在网页上展示虚拟宠物

## 功能说明

在网页上显示可爱的虚拟宠物，支持拖拽和动画效果。

## 功能分类

🐾 基础功能

## 使用方法

- **显示/隐藏**：`Ctrl+Shift+P`
- **移动**：按住左键拖动
- **位置自动记忆**：宠物位置会自动保存，下次打开网页时恢复

## 相关文件

- `modules/pet/content/petManager.pet.js`
- `modules/pet/content/petManager.drag.js`
- `modules/pet/content/petManager.state.js`

## 功能特点

- 支持多种宠物角色
- 流畅的拖拽体验
- 位置自动保存和恢复
- 动画效果
