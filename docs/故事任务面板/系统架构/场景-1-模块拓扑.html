YiPet Module Topology

全量模块识别 · 依赖图 · IIFE 全局命名空间 · 入口验证

Core Infrastructure

  • config.js — Configuration hub with all ENDPOINTS defined
  • token.js — TokenManager with 3-level fallback (env / storage / empty)
  • request.js — RequestClient with retry + timeout + X-Token injection
  • ApiManager.js — Interceptor chain for request/response
  • error.js — ErrorHandler with context invalidation detection

Pet Module

  • PetManager — 800+ line core class, single entry point
  • 10 content behavior modules (chat, drag, ui, state, events...)
  • 2 AI modules (ai.api.js + ai.prompt.js)
  • 4 session modules (CRUD, filter, tag, batch)
  • 9 Vue 3 components (ChatWindow through SessionTagManager)

Extension Backend

  • Service Worker entry (index.js) — persistent background
  • MessageRouter — dispatches 7 action categories to handlers
  • WeChat Work integration (WeWorkHandler)
  • Popup control panel (PopupController)
  • chrome.tabs.sendMessage bridges SW to content scripts
