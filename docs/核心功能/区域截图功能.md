# 区域截图功能

> 自由选择截图区域，便捷快速截图

---

## 功能概述

区域截图功能允许用户在网页上自由选择截图区域，并将截图直接插入到聊天中。

---

## 主要特性

### 1. 区域选择
- 自由拖拽选择截图区域
- 显示选区大小和位置
- 支持调整选框

### 2. 截图操作
- 一键完成截图
- 自动保存截图数据
- 支持取消操作

### 3. 集成功能
- 截图直接插入聊天输入框
- 支持发送截图消息
- 预览截图内容

---

## 使用说明

### 启动截图
1. 在聊天窗口中点击截图按钮
2. 或使用截图快捷键（如果配置）

### 选择区域
1. 在页面上按住鼠标左键
2. 拖动选择要截取的区域
3. 松开鼠标完成选择

### 完成操作
- **确认截图**：点击确认按钮或按 Enter
- **取消截图**：按 Escape 键或点击取消
- **调整区域**：拖动选框边缘调整大小

---

## 技术实现

### 相关文件
- `modules/pet/content/petManager.screenshot.js` - 截图功能实现
- `modules/screenshot/content/petManager.screenshot.js` - 截图核心逻辑
- `modules/extension/background/actions/screenshotHandler.js` - 后台截图处理

### 截图流程
1. 用户触发截图功能
2. 后台脚本获取当前标签页截图
3. 内容脚本显示截图遮罩层
4. 用户选择区域并确认
5. 截图数据传递到聊天输入框

### 数据格式
截图数据使用 Base64 编码：
```javascript
{
  dataUrl: 'data:image/png;base64,...',
  width: 800,
  height: 600,
  timestamp: Date.now()
}
```

## 配置选项

### 截图设置
- 图片格式：PNG / JPEG
- 图片质量：低 / 中 / 高
- 自动复制到剪贴板

### 快捷键配置
在 `manifest.json` 中配置：
```json
{
  "commands": {
    "take-screenshot": {
      "suggested_key": {
        "default": "Ctrl+Shift+S",
        "mac": "Command+Shift+S"
      },
      "description": "启动截图"
    }
  }
}
```
