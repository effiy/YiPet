概述

ChatWindow 是聊天窗口的根组件，130+ 方法管理整个聊天生命周期。它加载 ChatHeader/ChatInput/ChatMessages/FaqManager/FaqTagManager 子组件，管理窗口大小/位置/全屏/拖拽/主题/侧边栏/消息收发。

// 更新窗口样式（宽高位置）
updateChatWindowStyle() {
  const state = this.manager.chatWindowState || {}
  if (state.isFullscreen) {
    this.element.classList.add('fullscreen')
    return
  }
  this.element.style.setProperty('width', `${state.width || 850}px`)
  this.element.style.setProperty('height', `${state.height || 720}px`)
}
