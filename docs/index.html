业务场景

5 场景 · 3 模块层 · 10 Vue 组件
🐶 场景 1 · 页面宠物陪伴 — 注入 → 渲染 → 交互
▶
注入与初始化
Service Worker 监听标签页，通过 injectionService 向页面注入 content-script。PetManager.core 初始化宠物状态（位置/颜色/大小/角色），pet.js 渲染宠物 DOM，挂载到页面 body。
entry
background/index.js → petManager.core.js → pet.js
⚖
拖拽与定位
drag.js 处理鼠标/触摸拖拽事件，结合 domHelper 位置计算，支持页面内自由移动。位置变更通过 state.js 集中管理，经 messageHelper 跨标签页同步。
drag
drag.js + state.js + messageHelper
👁
外观与状态切换
ui.js 管理宠物动画和唤醒/睡眠态切换。颜色、大小、角色由 state.js 驱动，popup 或 ChatHeader 可远程控制。睡眠态降低页面干扰。
ui
ui.js + pet.js + state.js
💬 场景 2 · AI 对话聊天 — 窗口 → 输入 → 流式响应 → 渲染
W
聊天窗口与布局
ChatWindow 三栏布局（侧边栏 + 消息区 + 输入区），支持拖拽调整大小。chunk 持久化到 chrome.storage，重载后恢复完整界面。Hooks 模式注入 store/useComputed/useMethods 实现关注点分离。事件总线和 session/index.js 联动。
Vue 3
ChatWindow + ChatHeader + ChatWindowUtils
I
消息输入
ChatInput 自适应 textarea + CJK 输入法兼容 + 图片粘贴预览移除 + 7 按钮工具栏。editor.core.js 提供 Markdown 编辑内核，editor.ui.js 负责富文本交互和快捷键绑定。
Vue 3
ChatInput + editor.core.js + editor.ui.js
A
AI 流式请求
ai/api.js 通过 ApiManager 发起流式 SSE 请求，ai/prompt.js 构建 Prompt 上下文。Service Worker 的 messageForwardHandler 中继请求 → petHandler 协调 content ↔ background 通信。TokenManager 三级降级注入认证头。
stream
ai/api.js + chat.js + TokenManager + messageForwardHandler
M
消息渲染
ChatMessages 四视图状态 + 7 步渲染管线：输入 → marked.js 解析 → Mermaid 代码块检测 → CDN 懒加载 Mermaid → 图表渲染 → DOM 插入 → 智能滚动。chatUi.js 协调 UI 刷新。
Vue 3
ChatMessages + marked.js + renderer.js + chatUi.js
📊 场景 3 · 会话生命周期管理 — 创建 → 筛选 → 标签 → 同步
S
会话 CRUD
session.crud.js 依托 SessionService 实现完整的创建/读取/更新/删除。ChatHeader 展示当前会话标题和切换入口。新标签页自动创建会话，pageInfo.js 提取页面信息作为会话元数据。
crud
session.crud.js + SessionService + ChatHeader
F
筛选与标签
session.filter.js 按标题/标签搜索过滤，session.tag.js 管理标签关联，session.batch.js 批量操作（删除/归档）。SessionTagManager 支持拖拽排序标签，快速标签和手动输入。搜索过滤结果即时更新侧边栏。
Vue 3
SessionTagManager + session.filter.js
☁
数据同步
会话状态通过 storageUtils.js 持久化到 chrome.storage.local，跨标签页状态由 messageHelper 广播同步。配额超限时 StorageHelper.LRU 淘汰旧会话数据，保证写入不丢失。
sync
storageUtils.js + messageHelper + state.js
💡 场景 4 · FAQ 知识管理 — 条目 → 标签 → 搜索 → AI 辅助
F
FAQ 条目管理
FaqManager 组件树 + 简介卡片 + 内容列表 + 工具栏，6 视图状态（加载/空/错误/列表/搜索/详情），CRUD 操作通过 FaqService 与远端 API 同步。faq.js 独立模块提供页面级 FAQ 注入。
Vue 3
FaqManager + FaqService + faq.js
T
标签与 AI 辅助
FaqTagManager 标签编辑弹窗：快速标签一键添加 + 手动输入 + AI 智能生成标签建议。tags.js 独立模块管理标签 CRUD。robot.js 控制机器人自动回复行为。
Vue 3
FaqTagManager + tags.js + robot.js
⚙ 场景 5 · 扩展控制与认证 — 面板 → 认证 → 通知
P
控制面板
PopupController 管理宠物显示/隐藏、颜色、大小、角色切换。通过 messageRouter → tabMessaging → content-script 下发指令，state.js 统一更新。跨标签页状态自动同步，确保多页面一致。
popup
popup/index.js + messageRouter + tabMessaging
A
Token 与模型配置
TokenSettingsModal 密码模式输入 Token + 模型下拉选择 + 三级降级存储（环境变量 → storage → 提示设置）。AiSettingsModal 极简模型名称输入（107 行）。auth.js 管理认证状态和登录流程。
Vue 3
TokenSettingsModal + AiSettingsModal + auth.js
W
企业微信通知
weworkService.js 封装 Webhook 发送，weworkHandler.js 处理通知回调与消息模板。AI 回复完成后可选推送到企微群，messaging.js 管理通知生命周期。
notify
weworkService.js + weworkHandler.js + messaging.js
⚠
退化防御
errorHandler.js 全局 isContextInvalidated() 守卫 + isChromeStorageAvailable() 预检。扩展重载/上下文失效时优雅降级（隐藏宠物、清理 DOM、提示刷新）。所有 chrome API 调用经过预检，三层退化对策保护核心体验不崩溃。
defense
errorHandler.js + token.js + storageUtils.js
