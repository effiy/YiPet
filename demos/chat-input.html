ChatInput 使用事例

概述

ChatInput 管理聊天输入区域，包括 textarea 自动高度、Enter 发送、图片粘贴/上传、草稿图片管理、工具栏按钮（页面上下文/编辑会话/标签管理/FAQ/微信机器人/图片上传）、上下文开关、请求状态按钮。

输入框行为

键盘事件 · 自动高度 · 粘贴图片 · IME 组合输入

// Enter 发送 (Shift+Enter 换行)
onTextareaKeydown(e) {
  if (e.key === 'Escape') { 清空输入; blur }
  if (e.key === 'Enter' && !e.shiftKey) {
    if (isComposing) return  // 忽略 IME 组合中的 Enter
    instance.sendMessage()
  }
}
