## 在网页上看到虚拟宠物

**作为一个** 用户，**我想要** 在网页上看到可爱的虚拟宠物，**以便** 提升浏览体验，缓解压力。

**验收标准：**
- 宠物能够在网页上正确显示
- 宠物有可爱的外观和动画效果
- 宠物不会影响网页的正常使用

## 通过快捷键切换宠物显示

**作为一个** 用户，**我想要** 通过快捷键快速切换宠物的显示和隐藏，**以便** 方便地在需要时看到宠物，不需要时隐藏它。

**验收标准：**
- 使用 `Ctrl+Shift+P` (Mac: `Cmd+Shift+P`) 可以切换宠物显示/隐藏
- 切换操作响应迅速，无明显延迟
- 宠物状态变化有平滑的动画过渡

## 记住宠物的位置

**作为一个** 用户，**我想要** 宠物记住上次的位置，**以便** 下次打开网页时宠物出现在我喜欢的位置。

**验收标准：**
- 宠物位置自动保存到本地存储
